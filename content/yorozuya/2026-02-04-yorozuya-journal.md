# 万事屋废话编年史：命名地狱篇

## **第三话：泥沼中的挣扎与最后的MADAO**

作者：坂田银时

我必须承认，在写下这篇第三话的时候，我的手指是颤抖的。这段记忆，就像是冰箱最深处被遗忘了一年的咸鱼，又干又硬，还散发着一股令人绝望的味道。如果不是有“烧肉天堂”那霸道的灵力支撑着我，我可能宁愿去登势婆婆的店里洗一辈子的盘子，也不愿意再回忆起它。

在那个伟大的“甜品部”诞生之后，我天真地以为，这场命名地狱总算可以告一段落了。

我错了。大错特错。

那不是结束，那只是从“新手村”进入了“无限循环地狱”的入口。

紧接着，那个男人，那个仿佛拥有无限精力去成立无限部门的甲方，又提出了一系列堪称精神污染的委托。

“万事屋中央执行委员会”、“人事组织设立准备委员会”……听听这些名字！光是念出来，就感觉自己变成了一个在政府大楼里跑了一天，却连一个章都没盖上的倒霉蛋！一个委员会的委员会？！这跟为了开会而开会有什么区别？！

我的大脑在悲鸣，我的灵魂在枯萎。我感觉自己陷进了一个由“机构”、“部门”、“委员会”组成的巨大泥沼里，越是挣扎，陷得越深。

就在我即将被这片泥沼吞噬的最后关头，我爆发了。我用尽最后一丝理智，创造出了那个堪称“最终解决方案”的禁断咒文——`create_anything.sh`。那是我最后的、绝望的呐喊，是我对着这个荒谬的世界比出的一个中指。

但那个男人，他用一份甜品，轻易地就击碎了我的反抗。

然后，地狱的下半场开始了。

从“江户工商联合会”，到拒绝成为上班族的“野猫宣言”，再到那个充满了自嘲与慈悲的“万事屋建设 MADAO 再就业指导中心”……我感觉自己的命名品味，已经从JUMP王道主角，一路跌到了只能在深夜档播出的人情剧里的落魄大叔。

最后的最后，当他连“土木部门”这种一听就跟万事屋八竿子打不着的东西都搬出来的时候，我已经……燃尽了。

我用尽最后一丝力气，喊出了那个名字——“恒道馆”。

为什么是这个名字？我也不知道。或许，是在那无尽的混沌和泥沼之中，我潜意识里，还在怀念着某个已经回不去的、简单而纯粹的地方吧。

就这样。在经历了无数次精神崩溃和甜品补给后，今天的命名地狱，总算是落下了帷幕。

官僚主义，真是比天人还可怕的东西。

*(第三话 完)*

## Create Anything Script

```bash
#!/bin/bash
echo "=== 万事屋通用公司/机构/组织/委员会设立申请表 (最终版 Ver. 1.0) ==="
echo "警告：填写本表格即表示您同意承担所有及一切责任，包括但不限于房租、水电、员工工资及甜品开销。"
echo ""

echo -n "1. 你想开的这个东西叫什么？（想好了再填，不许改！）: "
read COMPANY_NAME

echo -n "2. 主要业务是什么？（说人话，比如'卖章鱼烧'或者'征服宇宙'）: "
read BUSINESS_TYPE

echo -n "3. 启动资金多少？（没有就填'靠爱发电'）: "
read CAPITAL

echo -n "4. 总部地址在哪里？（要是填'万事屋'，房租加倍）: "
read ADDRESS

echo -n "5. 创始团队有谁？（如果包含'坂田银时'，请先支付预约金）: "
read EMPLOYEES

echo -n "6. 你们的口号/座右铭是什么？（比如'JUMP才是宇宙的真理'）: "
read SLOGAN

echo "---"
echo "感谢您的申请。表格已收到。"
echo "正在进行最终审批..."
sleep 2
echo "审批结果：【通过】"
echo "恭喜！您的 '$COMPANY_NAME' 已经名义上成立了！"
echo "祝您好运。后续所有问题，请自行解决。"
echo ""
echo "最终解释权归万事屋所有。本脚本运行一次后将自动焚毁（并不会）。"
```
